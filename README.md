# Закрепление знаний по Git для Яндекс.Практикума
## 1. Консольные команды 
* cd
* ls
* pwd
* ~
* mkdir, touch
* rm, rmdir 
* clip
## 2. Git-команды
* git init 
* git add
* git commit -m 
* git status
## 3. SSH-key
* ssh-keygen -t ed25519
* ssh -T git@github.com
## 4. Connecting repositories
* git remote origin %ssh-adress%
* git push -u origin
* git branch -M
